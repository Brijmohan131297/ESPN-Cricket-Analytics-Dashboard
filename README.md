# 🏏 ESPN Cricket Analytics Dashboard

### Power BI | Web Scraping | Dynamic Parameters | Advanced DAX

---

## 🚀 Project Overview

The **ESPN Cricket Analytics Dashboard** is an end-to-end Power BI project built using dynamically scraped cricket statistics from the ESPN website.

This project demonstrates advanced Business Intelligence capabilities including:

- 🌐 Web scraping from live sports webpages
- ⚙️ Dynamic parameter-based data extraction
- 🧠 Advanced DAX calculations
- 🏗 Optimized data modeling (Star Schema)
- 📊 Interactive cricket performance analytics

This is a complete **Data Engineering + Analytics + Visualization** pipeline built in Power BI.

---

## 🎯 Project Objective

To build a scalable and automated cricket analytics solution that:

- Analyzes team performance
- Tracks player statistics
- Evaluates match-level KPIs
- Enables season-based filtering
- Uses dynamic web extraction without manual updates

---

## 🌐 Data Source

- **Source:** ESPN Cricket Section
- **Sport:** Cricket
- **Data Type:** Public match statistics, team stats, player metrics
- **Extraction Method:** Power Query (Web Connector)
- **Refresh Type:** Dynamic & Parameter-driven

> ⚠️ Disclaimer: All data belongs to ESPN. Used for portfolio and educational purposes only.
> 

## 🛠️ Tech Stack

## 📊 Cricket KPIs Analyzed

The dashboard provides insights such as:

1. Total Runs
2. Strike Rate
3. Batting Average
4. Bowling Economy
5. Wickets Taken
6. Team Win %
7. Match Performance Comparison
8. Seasonal Trends

## 🔥 Key Features

### 1️⃣ Dynamic Cricket Season Selection

Users can change season/year via parameter → which dynamically updates the ESPN URL → refreshes dataset.

### 2️⃣ Team & Player Analysis

- Team-level performance comparison
- Player ranking system
- Performance breakdown by match

### 3️⃣ Advanced DAX Logic

- Ranking players by runs/wickets
- Rolling averages
- Conditional KPIs
- Performance comparison metrics

### 4️⃣ Interactive Visual Experience

- Slicers (Team / Season / Player)
- Drill-through analysis
- KPI cards
- Trend charts
- Performance heat mapping
